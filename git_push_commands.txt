# Git Push Commands for Modern360 Assessment

## After creating a GitHub repository, run these commands:

# Add remote repository (replace YOUR_GITHUB_URL with your actual repository URL)
git remote add origin YOUR_GITHUB_URL

# Rename branch to main (GitHub standard)
git branch -M main

# Push to remote repository
git push -u origin main

## Example with a real URL:
# git remote add origin https://github.com/DzenanMuftic/modern360-assessment.git
# git branch -M main  
# git push -u origin main
